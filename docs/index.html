No Device Limitations

Can be accessed from any device and does not rely on any proprietary software so does not need expensive licensing or software.

Deployable

Hosted in a simple docker image, meaning different instances can be deployed for different projects or groups.

Secure

Access to the remote machine can be password protected. Can also be hosted locally [requires extra hardware], without access from off site.

Configuration

Variables that are constant such as youngs modulus and bioreactor heights are stored but can be modified if the system is changed

Asynchronous

Recording, tracking, and analysis are all separate tasks and can often be carried out at simultaneously on different sets of tissues. This makes it much more efficient than it having to be done linearly.
